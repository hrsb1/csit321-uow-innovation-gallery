# UOW Online Innovation Gallery (CSIT321)

A cloud-based web platform developed as part of CSIT321 at the University of Wollongong, designed to showcase student innovation projects.

## Tech Stack
- Next.js (React)
- AWS Amplify (Gen 2)
- AWS Cognito (Authentication)
- GraphQL (AppSync)
- DynamoDB & S3
- Tailwind CSS & Ant Design

## Features
- Student project submission and discovery
- Admin moderation and approval
- Secure authentication and role-based access
- Tag-based filtering and search

## Project Structure
- app/ – Application source code
- docs/ – Technical report, user manual, deployment guides
- assets/ – Poster and promotional materials
