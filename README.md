# UOW Online Innovation Gallery (CSIT321)

A cloud-based web platform developed as part of **CSIT321 – Software Development Project**
at the **University of Wollongong**, designed to showcase and manage student innovation projects.

## Tech Stack
- **Frontend:** Next.js (React), Tailwind CSS, Ant Design
- **Backend:** AWS Amplify (Gen 2), AWS AppSync (GraphQL)
- **Authentication:** AWS Cognito
- **Database & Storage:** DynamoDB, Amazon S3
- **Hosting:** AWS Amplify

##  Key Features
- Student project submission and discovery
- Secure authentication and role-based access
- Admin moderation and approval workflows
- Tag-based filtering and search
- Scalable serverless architecture

## Architecture Overview
The application follows a serverless cloud architecture using AWS Amplify to integrate
authentication, API, and data storage services with a modern React-based frontend.
